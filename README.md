# 📝 초등 5학년 국어 맞춤법 문제은행 생성기 (Korean Spelling Quiz Generator)

> **헷갈리기 쉬운 우리말 맞춤법과 어휘 문제를 엑셀(CSV) 파일로 자동 생성해주는 파이썬 도구입니다.** > 5~6학년 국어 교과 과정 및 받아쓰기 자료를 기반으로 제작되었습니다.

---

## 🧐 프로젝트 소개 (About)

초등학교 고학년이 되면 '좇다/쫓다', '껍데기/껍질', '지양/지향' 등 소리는 비슷하지만 쓰임새가 다른 어휘들이 등장합니다.
이 프로젝트는 이러한 **혼동 어휘(Confusing Words)**들을 퀴즈 형태로 풀 수 있도록 **문제은행 파일(.csv)을 생성**해줍니다. 생성된 파일은 엑셀이나 구글 스프레드시트에서 바로 활용할 수 있습니다.

### 🎯 주요 학습 내용

- **용언의 활용:** 맞히다 vs 맞추다, 낫다 vs 낳다
- **유사 어휘:** 껍데기 vs 껍질, 지양 vs 지향
- **표기법:** 으스대다, 구시렁거리다, 얽히고설키다 등

---

## ✨ 주요 기능 (Features)

1.  **자동 파일 생성**: 스크립트 실행 한 번으로 문제은행 파일(`5학년_국어_문제은행.csv`)을 생성합니다.
2.  **엑셀 호환성**: `UTF-8-SIG` 인코딩을 사용하여 엑셀에서 한글 깨짐 없이 바로 열람 가능합니다.
3.  **상세한 해설**: 정답뿐만 아니라, 해당 어휘가 왜 정답인지에 대한 친절한 문법적 해설을 포함합니다.
4.  **확장성**: `quiz_data` 리스트에 데이터를 추가하면 얼마든지 문제를 늘릴 수 있습니다.

---

## 🚀 사용 방법 (Usage)

### 1. 환경 설정 (Prerequisites)

Python 3.x 버전이 설치되어 있어야 합니다.

### 2. 실행 (Run)

터미널(또는 명령 프롬프트)에서 아래 명령어를 입력하세요.

```bash
python 5학년_국어_문제은행.py
```
